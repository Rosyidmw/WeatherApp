# WeatherApp

Aplikasi Pelacak Cuaca sederhana

## Clone Repository
```bash
git clone https://github.com/Rosyidmw/WeatherApp.git
```

## Mendapatkan API

```
1. Buka https://www.weatherapi.com/
2. Login hingga mendapatkan API
```

## Paste API

paste API pada file Constant.kt
```
object Constant {
    val apiKey = "your_api"
}
```
